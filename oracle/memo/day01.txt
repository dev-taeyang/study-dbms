설치 비밀번호 1234(sys system)

계정

	sys	: 모든 권한을 가지고 있는 계정.(너무많은 권한을 가지고 있음)
	system	: 계정 관리(적당한 권한을 가지고 사용하기위해. 안정성)
	일반 계정(hr) : 스키마(데이터를 볼수 있는 표나, 그래프 등이 나와있는 틀) 관리, 스키마스 안에 일반 계정들이 있음


스키마
	정리가 잘 되어있는 표들의 묶음, 상태

일반 계정 접속(hr)
 	  cmd 접속 > sqlplus system/1234
 	  SQL> show user
 	 USER is "SYSTEM"
 	  SQL> alter user hr account unlock;
  	 SQL> alter user hr identified by hr;
  	 SQL> conn hr/hr
 	  SQL> show user
 	  USER is "HR"
